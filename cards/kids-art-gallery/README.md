# 给孩子办一场线上画展  ·  An online gallery for your kid's art

> ✨ 帮你完成一个想法 · 难度：入门 · 适合：有孩子的家庭 · 约 4 个实验

![screenshot](./screenshot.png)

## 体验（先玩）
办一场实体画展几乎不可能、也很贵。把孩子的画拍下来，十分钟做一个**会发光、能旋转、带网址**的在线画廊；可配文字甚至配音，孩子能逐幅讲给别人听。

▶ Playground：https://aistudio.google.com
（生成式建站参考：LlamaCoder https://github.com/Nutlope/llamacoder ）

## 原理（它怎么工作）
这张卡的魔法不是“AI 画画”，而是 **AI 帮你写一个网页**（vibe coding）：

- 你用一句话描述想要的东西（“一个深色背景、能左右滑动、每张画配标题和日期的画廊”）。
- 大模型把这句话**翻译成前端代码**（HTML/CSS/JS 或 React），因为它读过海量网页代码，知道“画廊”长什么样、怎么写。
- 你把孩子画作的照片放进去，AI 再帮你加“灯光、旋转、放大”这些效果——每一个都是一小段它熟悉的代码。
- 最后一键部署，就有了一个**能发给亲友的网址**。

一句话：AI 把“我不会写网站”这件事，变成“我会描述我想要什么”。

## 你能学到什么
- 把照片/作品变成一个可分享的作品集（“把东西搬上网”的完整链路）
- 用自然语言迭代代码：不满意就继续说“再加个旋转 / 换个字体”
- 进阶效果（灯光、旋转、配音）其实是一个个可叠加的小模块

## 怎么复现（自己做）
1. **最快（无代码）**：打开 Google AI Studio → Build，用一句话描述你的画廊，让它生成一个 React 应用；把孩子画作的图片替换进去，导出/部署。
2. **看生成式建站**：`git clone https://github.com/Nutlope/llamacoder` —— 一句 prompt 生成一个可跑的小应用（开源 v0 思路），理解“描述→代码→预览”的闭环。
3. **加进阶效果**：让 AI 给你加“图片轮播 / 点击放大 / 背景音乐（配音）”，每次只说一个需求。
4. **上线 + 配音**：部署到任意静态托管（如 Cloudflare Pages）拿到网址；配音可用浏览器录音或文本转语音，让孩子讲每幅画的故事。
5. **需要**：几张画作照片 + 一个能生成网页的 AI（AI Studio / LlamaCoder）+ 一个免费托管。
